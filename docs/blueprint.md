# **App Name**: Block Party

## Core Features:

- Game Grid: Generate and display the Tetris grid.
- Piece Generation: Randomly generate Tetris pieces.
- Piece Movement: Enable movement and rotation of the current piece.
- Collision Detection: Detect collisions and lock pieces in place.
- Line Clearing: Detect and clear completed lines, award points, and accelerate the game, using the rules of Tetris.
- Game Over: Determine game over state when pieces reach the top.
- Score Tracking: Track the player's score, updating it upon line clears.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0), inspired by the nostalgic, playful feeling of classic arcade games.
- Background color: Dark gray (#222222), same hue as primary but highly desaturated to give contrast for other colored objects and UI elements to stand out.
- Accent color: Electric blue (#7DF9FF), providing high contrast for active elements and highlights.
- Font: 'Space Grotesk' sans-serif for a modern yet blocky feel suitable for both titles and instructions.
- Use simple, geometric icons to represent controls.
- Clean and minimal layout with the game grid centered on the screen.
- Smooth transitions for piece movements and line clearing effects.
